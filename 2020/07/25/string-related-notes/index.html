String相关知识点笔记

Posted on 2020-07-25

1. String

1.1 特点

  • 使用fianl修饰，不能被继承
  • 底层是final修饰的char[]数组，不能被修改

1.1 String常量池

  • Java使用常量池机制，将字符串常量放入常量池中以实现复用

  • 当使用""直接给字符串赋值时，使用的是常量池中的字符串，如果给两个对象赋值同样的字符串常量，则两者的地址相同

  • 当使用new关键字构造String对象时，创建新的对象，指向一个全新的内存地址而不是常量池中的任何一个常量的地址

  • 使用intern()方法时，如果常量池中有该调用对象，则直接返回该对象的地址，否则向常量池中添加这个对象，并返回对它的引用

2. StringBuilder和StringBuffer

2.1 共同点

  • 和String一样，使用final修饰，不能被继承

2.2 区别

  • StringBuilder是非线程安全的，而StringBuffer是线程安全的（所有的公开方法都是synchronized修饰的）

  • StringBuffer有缓存区，每次调用toString()方法时，都会直接使用缓存区的值来构造一个字符串

    而StringBuilder则在每次调用toString()方法时都复制一次字符数组来构造一个字符串
